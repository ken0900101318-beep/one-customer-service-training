## 📋 內容涵蓋

### 1. 基本資料
- 公司資訊
- 客服時間
- 服務重點與規則
- 環境規則（外食、電視設備）
- 優惠活動

### 2. 全台分店資料（13家）
- 📍 新北：土城亞東店、新莊輔大店
- 📍 桃園：平鎮店
- 📍 台中：一中店、逢甲無老板店、民權店
- 📍 雲林：虎尾店
- 📍 嘉義：光彩店、公園店
- 📍 台南：永康崑大店
- 📍 高雄：前鎮店、仁武店

每間店包含：
- 地址、預約網址
- 飲料機/飲水機資訊
- 價格表（含早鳥優惠）
- Wifi密碼
- Google評論連結

### 3. 常見問題處理
- 💰 金錢爭議（卡幣、重複扣款）
- 📞 改單、取消與退費
- 🔐 進場密碼與電力問題
- 🧹 環境髒亂問題
- 🀄 麻將桌故障排除
- 🪑 椅子損壞
- 🎁 續時券領取
- 🏪 非本店店家處理

### 4. 商家端後台操作
- 更換包廂
- 取消訂單/退款
- 查詢客人訂單
- 處理金錢爭議
- 補發續時券
- 緊急聯絡方式
